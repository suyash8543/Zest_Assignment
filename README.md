🎓 Student Management System API

A full-stack backend project built using "ASP.NET Core Web API" with "Entity Framework Core", "SQL Server", and "JWT Authentication".

---

 🚀 Features

- ✅ Get all students
- ✅ Add new student
- ✅ Update student
- ✅ Delete student
- ✅ JWT Authentication (Secure APIs)
- ✅ Global Exception Handling
- ✅ Logging (Serilog / Built-in)
- ✅ Swagger API Documentation

---

 🏗️ Tech Stack

- ASP.NET Core Web API (.NET 9)
- Entity Framework Core
- SQL Server
- JWT Authentication
- Swagger (Swashbuckle)
